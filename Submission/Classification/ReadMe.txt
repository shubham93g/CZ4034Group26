**************Procedure for Classification*****************

1. Run SVM classifier.py
2. The predicted output is stored in predicted output.csv
3. Run evalTopLevel.py, which compares predicted output with the gold file and displays classifier performance statistics. 



**************Description of CSV data files***************

1. training data.csv - contains 1600 records with class labels used for training
2. test data.csv - contains 200 records used for testing the classifier
3. test data(gold).csv - contains test data with the class labels. 
4. predicted output.csv - contains the predicted class labels for the classifier.
